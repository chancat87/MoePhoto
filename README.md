### 基本功能[Image/Video]

- 降噪 Denoising
- 超分辨率 Super resolution
- 去摩尔纹 Demoireing
- 去雾 Dehaze
- 涂抹及风格化 Force denoise
- 高帧率和慢动作 High FPS and Slow motion

### 硬件需求

- CPU 模式 (CPU mode)
> 系统内存大于 4GB
> RAM > 4GB
- GPU 模式 (GPU mode)
> 仅支持 NVIDIA 显卡(>=GTX 750Ti)
> 显存不小于 2GB
> Only support NVIDIA cards
> VRAM should > 2GB
